Sales Analysis

Features

This project analyzes sales performance of the market over the following aspects:

  1. General Sale Performance
  2. Category-wise Sale Performance
  3. Customer-wise Sale Performance

The analysis provides aggregate data and graphs based on the provided data, which are useful for companies to categorize and identify strengths and weaknesses in their business.

Libraries Used

  • Python
  • Pandas
  • NumPy
  • GraphMaker
  • DataHandler
